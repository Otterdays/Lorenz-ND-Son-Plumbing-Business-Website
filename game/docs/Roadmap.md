# Implementation Roadmap

Milestone 0 — Concept (1–2 days)
- Finalize GDD, wireframes, scoring, event schema

Milestone 1 — Prototype (2–3 days)
- Build grid, rotation, path validation; barebones UI

Milestone 2 — Polish (2–4 days)
- Add pressure/leak mechanics, score HUD, sound, haptics

Milestone 3 — Brand Integration (1–2 days)
- Apply colors/fonts; add CTAs and post-round panel

Milestone 4 — Accessibility & Performance (1–2 days)
- Keyboard controls, ARIA, reduced motion; asset compression; lazy audio

Milestone 5 — Gatsby Integration (1 day)
- Route `/game`; analytics event hooks; track CTA clicks

Milestone 6 — QA & Launch (1–2 days)
- Browser/device testing; performance thresholds; publish

Milestone 7 — Post-Launch Iterations (ongoing)
- Difficulty tuning, add levels; A/B test CTA placement
