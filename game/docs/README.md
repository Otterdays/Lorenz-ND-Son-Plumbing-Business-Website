# Plumbing Mini-Game: Brand-Aligned Concept

This documentation describes a lightweight, web-based mini-game designed for Lorenz & Son Plumbing. The game is fast-loading, mobile-friendly, and thematically aligned with plumbing services and the brand’s emergency-response positioning.

- Audience: Residential homeowners and commercial facility managers in Greater Chicago
- Brand: Navy `#0B1E3F`, Gold `#C89B3C`, Cream `#F9F4EA`; headings Playfair Display; body Source Sans Pro
- Business goals: Engage visitors, lower bounce rate, reinforce expertise, nudge emergency call and quote CTAs

Quick Pitch
- “Emergency Pipe Link” — quickly rotate pipe tiles to connect the water main to the destination before pressure builds and leaks appear. Simple tap/rotate controls, short rounds (30–60s), branded visuals, and subtle CTA placement.

Key Traits
- Lightweight: Vanilla JS/TS + CSS; small inline SVG icons; <50KB JS budget
- Mobile-first: Thumb-friendly controls; portrait 12–14 column grid
- Accessible: Keyboard controls, high-contrast colors, ARIA/roles, reduced motion
