## Quick Start (Windows 11)

1. Install Node.js 18+ and enable PowerShell execution for scripts if prompted.
2. Install dependencies:
   ```powershell
   npm install
   ```
3. Run the Gatsby dev server:
   ```powershell
   npm run develop
   ```
4. Visit `http://localhost:8000` to explore the staging build. Press `Ctrl+C` to stop.

## Project Highlights

- Hero section with logo, dual CTAs, and embedded Chicagoland service-area map.
- Split services layout for residential and commercial audiences.
- Sticky 24/7 emergency widget with response guarantee.
- Credential-rich about section, trust bar, and six-segment testimonial carousel.
- Dual contact forms (residential + commercial) with file upload support and Formspree placeholders.

## Tech Stack

- Gatsby 5, React 18, TypeScript
- Tailwind CSS with custom Lorenz & Son palette (deep navy `#0B1E3F`, antique gold `#C89B3C`, cream `#F9F4EA`)
- @fontsource Playfair Display & Source Sans Pro typography
